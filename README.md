# 📁 Media Organizer (macOS)

A modern, high-performance media organization tool designed for macOS. 

![Media Organizer Icon](Media%20Organizer/Assets.xcassets/AppIcon.svg)

## ✨ Features
- **Liquid Glass UI:** Built with a premium, translucent design language.
- **Spectrum Navigation:** Unique color-coded tabs for intuitive media sorting.
- **Smart Organization:** Automatically group and rename your media files.

## 🚀 Installation (Drag & Drop)
To install Media Organizer on your Mac:

1.  Download the latest installer: [Media-Organizer-Alpha-1.0.dmg](Releases/Media-Organizer-Alpha-1.0.dmg)
2.  Open the DMG file.
3.  **Drag and drop** the `Media Organizer` icon into your **Applications** folder.

## 🎨 Design Philosophy
The application follows a "Liquid Glass" aesthetic, featuring:
- **70-80% Translucency** for a frosted glass effect.
- **Unique Spectrum:** Electric Indigo, Tropical Cyan, Aurora Green, Solar Gold, and Sunset Magenta.
- **Specular Highlights:** Sharp, glassy reflections that respond to your system theme.
