# 📁 Media Organizer (macOS)

A high-performance media organization tool designed for macOS that leverages Artificial Intelligence to keep your workspace clean.

## 🚀 Key Features

### 🧠 AI-Powered Analysis
- **Intelligent Renaming:** Uses Llama 3.2 (local via Ollama) or OpenAI to analyze file content and filenames, proposing descriptive, human-readable names.
- **Contextual Categorization:** Automatically identifies the "Category Folder" (e.g., *Receipts, Taxes, Work Documents*) based on the file's data.
- **PDF Intelligence:** Capable of reading text inside PDF documents to understand their context.

### 🎵 Deep Media Metadata
- **ID3 Tagging:** Automatically writes Artist, Title, and Album metadata directly into your media files (.mp4, .m4a).
- **Format Support:** Comprehensive support for PDF, MP4, MOV, M4A, MP3, and WAV.

### 🏷️ Native macOS Integration
- **Finder Tags:** Automatically applies native macOS Finder Tags based on the AI-generated categories, making your files instantly searchable in Spotlight.
- **Drag & Drop Workflow:** A simple, high-performance interface built for power users.

## 🛠️ Installation

1.  Download the latest installer: [Media-Organizer-Alpha-1.0.dmg](Releases/Media-Organizer-Alpha-1.0.dmg)
2.  Open the DMG file.
3.  **Drag and drop** the `Media Organizer` icon into your **Applications** folder.

## ⚙️ Configuration
The app supports multiple AI backends. You can configure your endpoint and API keys in the Settings view to switch between local privacy (Ollama) and high-performance cloud models (GPT-4).
